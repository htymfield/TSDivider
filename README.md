# TSDivider

フォルダ内にたまった動画ファイルを一括でフォルダ分けするためのスクリプトです。ファイル名のみに基づいてフォルダを分類します。

SCRenameでフォルダ分けできなかったファイルをフォルダ分けすることを目的としています。

放映クール終了時のような、ひとつのフォルダ内に同じシリーズの動画が複数話たまっている状態を想定しています。

## 使い方
Visual Studioなどからプロジェクトの発行を行ってください。生成されたTSDivider.exeにフォルダをドラッグアンドドロップすると、フォルダ内のファイルが分類されます。

コマンドラインから実行する場合はオプションをつけることができます。コマンドは下記のとおりです。

`TSDivider.exe 対象フォルダ [オプション]`

## オプション

### --dry-run
ファイルの移動を行わなくなります。

### --copy
ファイルの移動を行わず、ファイルのコピーを行います。


## 対象とするファイル

SCRenameで推奨される形式と同じファイル名としてください。すなわち、ファイル名を以下の形式としてください。

`YYYYMMDDhhmm_番組名 _放送局名.拡張子`
